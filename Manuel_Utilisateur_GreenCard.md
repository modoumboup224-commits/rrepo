## 1. Introduction {#introduction}

### Qu'est-ce que GreenCard ?
GreenCard est une plateforme de e-commerce locale qui connecte directement les consommateurs avec les producteurs locaux. Elle permet d'acheter des produits frais, locaux et de saison tout en soutenant l'économie locale.

### Objectifs de la plateforme
- Faciliter l'accès aux produits locaux
- Soutenir les producteurs de la région
- Réduire l'empreinte carbone liée au transport
- Promouvoir une consommation responsable

## 2. Guide pour les Consommateurs {#consommateurs}

### 2.1 Création d'un compte

#### Étape 1 : Accès à l'inscription
1. Rendez-vous sur la page d'accueil GreenCard
2. Cliquez sur "S'inscrire" dans le menu principal
3. Choisissez "Consommateur"

#### Étape 2 : Formulaire d'inscription
- **Email** : Votre adresse email valide
- **Mot de passe** : Minimum 8 caractères avec majuscule, minuscule et chiffre
- **Nom complet** : Votre nom complet pour les commandes
- **Adresse** : Votre adresse de livraison
- **Téléphone** : Numéro pour les notifications de commande

#### Étape 3 : Validation
- Vérifiez votre email (lien de confirmation envoyé)
- Connectez-vous avec vos identifiants

### 2.2 Navigation sur la plateforme

#### Page d'accueil
- **Carrousel produits** : Découvrez les nouveautés
- **Catégories** : Fruits, légumes, produits laitiers, etc.
- **Producteurs vedettes** : Découvrez vos producteurs locaux

#### Recherche de produits
1. **Barre de recherche** : Tapez le nom du produit
2. **Filtres disponibles** :
   - Par catégorie
   - Par producteur
   - Par prix
   - Par disponibilité
   - Par distance

### 2.3 Processus d'achat

#### Ajouter au panier
1. Cliquez sur un produit pour voir les détails
2. Sélectionnez la quantité souhaitée
3. Cliquez sur "Ajouter au panier"
4. Le panier s'affiche en haut à droite

#### Finaliser la commande
1. **Panier** : Vérifiez vos articles
2. **Livraison** : Choisissez le mode de livraison
   - Retrait chez le producteur
   - Livraison à domicile
   - Point relais
3. **Paiement** : Sécurisé via Stripe
   - Carte bancaire
   - PayPal
4. **Confirmation** : Email de confirmation envoyé

### 2.4 Suivi des commandes

#### Tableau de bord
- **Commandes en cours** : Statut en temps réel
- **Historique** : Toutes vos commandes passées
- **Détails** : Produits, quantités, prix total

#### Notifications
- Email à chaque changement de statut
- SMS optionnel pour les livraisons

## 3. Guide pour les Producteurs {#producteurs}

### 3.1 Inscription Producteur

#### Conditions requises
- Être producteur local (rayon 50km)
- Produits frais ou transformés locaux
- Certifications si nécessaires (bio, AOP, etc.)

#### Processus d'inscription
1. **Demande d'inscription** : Formulaire détaillé
2. **Validation** : Vérification des documents
3. **Formation** : Tutoriel de prise en main
4. **Activation** : Compte activé sous 48h

### 3.2 Gestion du catalogue

#### Ajouter un produit
1. **Informations générales** :
   - Nom du produit
   - Description détaillée
   - Catégorie
   - Prix unitaire
2. **Images** :
   - Minimum 3 photos de qualité
   - Format JPG/PNG recommandé
   - Taille max 5MB
3. **Stock** :
   - Quantité disponible
   - Unité de mesure (kg, pièce, litre)
4. **Disponibilité** :
   - Dates de disponibilité
   - Jours de récolte/livraison

#### Gestion des stocks
- **Mise à jour en temps réel**
- **Alertes stock faible**
- **Historique des ventes**

### 3.3 Gestion des commandes

#### Tableau de bord producteur
- **Commandes reçues** : Nouvelles commandes
- **À préparer** : Produits à préparer
- **En livraison** : Commandes en cours
- **Terminées** : Commandes livrées

#### Processus de traitement
1. **Notification** : Email/SMS à chaque nouvelle commande
2. **Préparation** : Préparer les produits
3. **Validation** : Marquer comme "prêt"
4. **Livraison** : Remise au client ou livreur

### 3.4 Paiements et facturation

#### Cycle de paiement
- **Paiement client** : Immédiat via plateforme
- **Commission** : 10% retenue par GreenCard
- **Versement** : Tous les 15 jours
- **Facturation** : Mensuelle détaillée

## 5. Dépannage {#depannage}

### 5.1 Problèmes de connexion

#### Mot de passe oublié
1. Cliquez sur "Mot de passe oublié"
2. Entrez votre email
3. Suivez le lien reçu
4. Créez un nouveau mot de passe

#### Compte bloqué
- Contactez le support via le formulaire
- Vérification d'identité requise
- Déblocage sous 24h

### 5.2 Problèmes de commande

#### Commande non reçue
1. Vérifiez le statut dans votre tableau de bord
2. Contactez le producteur via messagerie
3. Si non résolu : support@greencard.fr

#### Produit non conforme
- Photos du produit reçu
- Description du problème
- Remboursement ou échange possible

### 5.3 Problèmes techniques

#### Site lent ou inaccessible
- Vider le cache du navigateur
- Essayer un autre navigateur
- Vérifier votre connexion internet

#### Erreur de paiement
- Vérifier les informations bancaires
- Contacter votre banque
- Essayer un autre moyen de paiement

## Glossaire

- **Producteur** : Agriculteur ou artisan local vendant sur la plateforme
- **Consommateur** : Personne achetant des produits sur GreenCard
- **Panier** : Sélection temporaire de produits avant commande
- **Commande** : Achat finalisé avec paiement
- **Commission** : Pourcentage retenu par GreenCard sur les ventes
- **Unité** : Quantité de base pour un produit (kg, pièce, litre)
